# 7GUIs

This repository implements the [7GUIs benchmark](https://eugenkiss.github.io/7guis/)—a set of seven tasks representing typical GUI programming challenges.

## The Seven Tasks

Each task demonstrates a specific GUI programming challenge:

1. **Counter**: Increment/decrement a counter button—demonstrates basic state management and event handling
2. **Temperature Converter**: Convert between Celsius and Fahrenheit—shows bidirectional data binding and synchronization
3. **Flight Booker**: Book flights with date validation—demonstrates form validation and state-dependent UI
4. **Timer**: Countdown timer—shows periodic updates and time-based state changes
5. **CRUD**: Create, read, update, delete items—demonstrates list management and CRUD operations
6. **Circle Drawer**: Draw and move circles—shows interactive graphics and mouse/drag handling
7. **Cells**: Grid-based cell simulation—demonstrates 2D grid state and complex interactions

For detailed specifications and requirements, visit the official [7GUIs website](https://eugenkiss.github.io/7guis/).
